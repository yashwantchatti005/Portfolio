# 🚀 Yashwant Chatti — Space Theme Portfolio

Welcome to my personal portfolio website!  
This is a fully responsive, interactive **space-themed developer portfolio** built to showcase my skills, projects, experience, certifications, and achievements as a Full Stack Developer Trainee.

🔗 **Live Demo:**  
https://yashwantchatti005.github.io/Portfolio/

---

## 🌌 About

I am a BCA student specializing in Full Stack Web Development.

I enjoy building modern, immersive, and performance-focused web applications.  
This portfolio represents my technical journey — designed with a sleek **space-inspired UI**, smooth animations, and a professional layout.

The website is built using pure HTML, CSS, and JavaScript and is deployed using GitHub Pages.

---

## ✨ Features

- 🌌 Space-themed UI with dark cosmic background
- 🌠 Glowing accents and smooth animations
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🍔 Animated hamburger navigation
- ✨ Scroll-triggered animations
- 📊 Skills section with animated progress bars
- 🚀 Project showcase with modal details
- 📧 Contact form integrated with EmailJS
- ⬆️ Smooth back-to-top button
- ⚡ Lightweight and fast loading

## 🚀 Installation (Local Development)

If you want to run this locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yashwantchatti005/Portfolio.git
```

### 2️⃣ Navigate into the folder

```bash
cd Portfolio
```

### 3️⃣ Open in browser

Simply open `index.html`  
Or use the Live Server extension in VS Code.
